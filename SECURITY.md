# Security Policy

We take security seriously.

## Reporting a Vulnerability
If you discover a security issue, please report it privately to:
**security@creatorshub.global**

Do not create a public issue.

We will:
- Acknowledge receipt within 48 hours
- Investigate and respond within 7 days
- Credit contributors responsibly (if desired)

## Scope
This applies to:
- All code in this repository
- The deployed platform at https://CreatorsHub.global
